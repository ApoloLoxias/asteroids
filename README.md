#Asteroids

Asteroids game coded in python3 with pygame library for boot.dev guided project

## Releases
### [V1.0 for windows](https://github.com/ApoloLoxias/asteroids/releases/tag/compiled_for_windows/asteroids_windows_1.0.exe)

### [V2.0 for windows](https://github.com/ApoloLoxias/asteroids/releases/tag/v2_compiled_for_windows)
- Improvements for player movement
- Projectile velocity now relative to player
- Asteroids boucne
- Player appears on the other side when leaving the screen

## Controls
WASD - movement (backward moviemtn with S key disabled for V2.0)
spacebar - shoot
